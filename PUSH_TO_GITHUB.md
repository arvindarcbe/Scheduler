# Push to GitHub - Instructions

## Option 1: Use the Script (Easiest)

1. Double-click `push_to_github.bat`
2. Enter your GitHub username when prompted
3. Done!

## Option 2: Manual Commands

Replace `YOUR_USERNAME` with your actual GitHub username and run:

```bash
git remote add origin https://github.com/YOUR_USERNAME/Scheduler.git
git branch -M main
git push -u origin main
```

## What is your GitHub username?

Your GitHub username is different from your display name "Aravind R". 

To find it:
1. Go to https://github.com
2. Click on your profile picture (top right)
3. Your username is shown in the URL or profile

Example: If your profile URL is `https://github.com/aravind123`, then your username is `aravind123`

## After Pushing

Your repository will be available at:
`https://github.com/YOUR_USERNAME/Scheduler`
